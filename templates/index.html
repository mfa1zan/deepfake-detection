Deepfake Detection Using

Neural Networks

This project detects deepfake videos using a hybrid CNN-LSTM neural network. We utilized the FaceForensics++ dataset, extracting facial landmarks from video frames with dlib and OpenCV. Each video was processed into sequences of 68 (x, y) facial landmarks per frame, normalized and organized for model input. The model architecture combines convolutional layers to capture spatial patterns within each frame and LSTM layers to learn temporal changes across frames. The system was trained and validated on carefully split train, validation, and test sets. We evaluated model performance using accuracy, confusion matrix, precision, recall, and F1-score, demonstrating robust detection of manipulated content. A user-friendly Flask web interface allows uploading new videos for real-time analysis and prediction. The pipeline delivers reliable deepfake detection to help combat video-based misinformation and manipulation.
